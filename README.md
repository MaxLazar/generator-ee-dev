# generator-ee-addon

An ExpressionEngine Add-On generator for [Yeoman](http://yeoman.io). Capable of building modules, plugins, extensions, accessories and fieldtypes.

### Prerequisites

Install Node.js

```
brew install node
```

Install `yo`

```
npm install -g yo
```

### Installation

```
$ npm install -g generator-ee-dev
```

### Usage

```
$ yo ee-dev
```
